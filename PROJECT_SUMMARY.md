## 📝 使用步骤

### 1. 安装 LaTeX

请参考 `INSTALL.md` 文件：

**macOS:**
```bash
# 下载安装 MacTeX
# 访问 https://www.tug.org/mactex/
```

**Windows:**
```bash
# 下载安装 MiKTeX
# 访问 https://miktex.org/download
```

**Linux:**
```bash
sudo apt install texlive-xetex texlive-lang-chinese
```

### 2. 编译模板

**方法1：使用 Makefile（推荐）**
```bash
cd /path/to/wordtolatex
make all
```

**方法2：手动编译**
```bash
xelatex xjtujournal_template.tex
xelatex xjtujournal_template.tex  # 第二次生成引用
```

**方法3：使用编辑器**
- 打开 `xjtujournal_template.tex`
- 选择 XeLaTeX 编译器
- 点击编译

### 3. 查看结果

编译成功后会生成 `xjtujournal_template.pdf`。

查看PDF：
```bash
make view
# 或手动打开 PDF 文件
```

### 4. 修改为您的论文

1. 复制模板：
```bash
cp xjtujournal_template.tex my_paper.tex
```

2. 打开 `my_paper.tex`，修改：
   - 期刊信息（卷期、年月）
   - 标题（中英文）
   - 作者信息
   - 摘要和关键词
   - 正文内容
   - 参考文献

3. 编译您的论文：
```bash
xelatex my_paper.tex
xelatex my_paper.tex
```

## ⚠️ 注意事项

1. **必须使用 XeLaTeX 编译**
   - PDFLaTeX 不支持中文
   - LuaLaTeX 也可以，但推荐 XeLaTeX

2. **编译两次**
   - 第一次生成内容
   - 第二次生成引用和页码

3. **中文字体**
   - 确保系统已安装宋体、黑体、楷体
   - macOS 用户可能需要额外安装字体

4. **图片格式**
   - 支持 PNG、JPG、PDF
   - 文件名不要包含中文和空格

5. **表格格式**
   - 使用三线表（booktabs）
   - 列格式：c（居中）、l（左对齐）、r（右对齐）
